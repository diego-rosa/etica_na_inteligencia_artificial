# Fio da Meada: Reflexões com Nina da Hora

Este é um site simples desenvolvido para refletir sobre questões éticas na inteligência artificial, baseado no episódio do podcast "Fio da Meada" com Nina da Hora.

## Como visualizar

Basta abrir o arquivo `index.html` em qualquer navegador web moderno.

## Conteúdo

O site aborda:
- A troca de privacidade por conveniência.
- O racismo algorítmico e falhas no reconhecimento facial.
- A importância de ler os termos de uso ("letras miúdas").
- A necessidade de regulação ética na tecnologia.

## Estrutura

- `index.html`: Estrutura e conteúdo.
- `style.css`: Estilização (Tema escuro/Cyberpunk).
- `script.js`: Interatividade (Modal de "letras miúdas").
